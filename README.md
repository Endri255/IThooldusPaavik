# IT Hoolduspäevik

See on Pythoni rakendus IT-hooldustööde ja remontide logimiseks. 
Projekt toetab nii käsurida (CLI) kui ka graafilist liidest (GUI). Andmed salvestatakse `logbook.json` faili.

## Funktsionaalsus
1. Kirjete lisamine (Pealkiri, Kirjeldus)
2. Kirjete vaatamine, otsimine ja filtreerimine
3. Staatuse muutmine (OPEN <-> DONE)
4. Kirjete kustutamine
5. Andmete importimine (vigade kontrolliga)

## Käivitamine
Veendu, et sul on Python 3 paigaldatud.
Käivita terminalist:
`python main.py`

Vali alguses, kas soovid kasutada CLI või GUI versiooni.

## Muutatused
Muutsin valikuid reast, tegin korda 6 ja 7 valiku korda, eet see nüüd töötaks.
